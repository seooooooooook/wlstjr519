1. 자바스크립트 코드는 작성된 순서대로 구문을 분석합니다. 만약에 코드에 문제가 생기면 에러가 발생한 다음 줄 부터는 구문을 분석하지 않습니다. 대부분의 브라우저에는 자바스크립트 코드를 컴파일 하는 JIT가 내장되어 웹 애플리케이션을 구현 할 수 있습니다.
2. 자바스크립트는 서버에서 실행되지 않고 사용자 컴퓨터에서 실행되는 클라이언트 언어입니다. 서버에서 운용되는 PHP, JSP, ASP 언어와 비교됩니다.
3. C++, JAVA와 같이 클래스를 이용하여 객체를 생성하는 클래스 기반 객체 지향 언어입니다. 자바스크립트는 클래스가 아닌 프로토타입을 상속하는 프로토타입 기반 객체 자향 언어입니다.
4. 자바스크립트를 이용한 언어에는 다양한 라이브러리 언어들이 있습니다. jQuery, node.js, React.js, Vue.js
5. C++, JAVA는 실행되기 전에 변수 타입을 정하는 언어입니다. 자바스크립트는 변수 타입이 없는 동적 타입의 언어이며, 자바스크립트 함수는 클로저를 정의합니다. 클로저는 변수를 숨기거나 영속성을 보장하는 다양한 기능을 구현합니다.
